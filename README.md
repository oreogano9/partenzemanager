# Partenze Manager

React + Vite app for managing departure loading tasks across terminal views, flight urgency, and unit-level baggage workflow.

## Stack

- React
- Vite
- Tailwind CSS
- lucide-react

## Run

```bash
npm install
npm run dev
```

## Notes

- The initial data is embedded in [`src/App.jsx`](/Users/konradparada/Documents/Codex/partenze_manager/src/App.jsx).
- Google Calendar and `.ics` export are available from expanded flight cards.
- The provided UI code was normalized into a buildable Vite app and a few missing translation labels were added.
